Jawaban Test 1

  1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework?
    Karena framework menyediakan kerangka kerja yang kokoh dan terstruktur untuk pengembangan perangkat lunak, mempercepat proses pengembangan, meningkatkan kualitas kode.
  2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?
    If else: untuk mengeksekusi satu blok kode jika kondisi tertentu terpenuhi,
    While: mengeksekusi blok kode selama kondisi tertentu masih terpenuhi,
    While do: melakukan perulangan dulu kemudian baru memeriksa syaratnya,
    Switch case: digunakan untuk memilih salah satu dari beberapa blok kode untuk dieksekusi berdasarkan nilai dari ekspresi,
    For: digunakan untuk mengulangi blok kode sejumlah tertentu kali, atau untuk setiap elemen dalam koleksi data,
    Foreach: digunakan untuk mengulangi elemen dalam koleksi data, seperti array atau daftar.
  3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?
    Constructor: Metode khusus dalam sebuah kelas yang otomatis dipanggil saat objek dari kelas tersebut dibuat, digunakan untuk inisialisasi,
    Method: Fungsi atau prosedur yang terkait dengan suatu objek atau kelas, digunakan untuk memanipulasi data atau melakukan operasi tertentu,
    Class: Blueprint untuk menciptakan objek,
    Object: instance nyata dari sebuah class,
    Variable: Lokasi penyimpanan untuk nilai dalam program,
    Parameter: Nilai yang diterima oleh sebuah fungsi atau metode saat dipanggil,
    Function: blok kode yang dapat melakukan tugas tertentu dan mengembalikan nilai,
    Procedure: Blok kode yang melakukan tugas tertentu tanpa mengembalikan nilai.
  4. Perbedaan method=”Post” dan “Get” ?
    POST: mengirim data dari klien ke server dalam tubuh permintaan HTTP,
    GET: mengirim data melalui URL.
  5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?
    Web service adalah sebuah layanan yang disediakan oleh suatu sistem internet, yang memungkinkan sistem lain untuk berkomunikasi dengannya dan menggunakan fungsionalitas yang ditawarkan. Fungsi yang digunakan untuk memanggil web service client adalah fungsi "HTTP Request".
  6. Jelaskan mengenai MVC dan HMVC !
    MVC membagi aplikasi menjadi model, view, dan controller, sementara HMVC memperluas konsep dengan struktur hierarkis yang memungkinkan modularitas lebih besar.
  7. Sebutkan fungi curl() dalam php!
    Digunakan untuk melakukan pengiriman permintaan HTTP ke URL tertentu dan memungkinkan untuk berinteraksi dengan server lain dan mengambil atau mengirim data melalui protokol HTTP.
  8. Bagaimana menampilkan sebuah element array dalam PHP ?
    Menggunakan perulangan foreach untuk array indeks atau array asosiatif.
  9. Perbedaan HTTP dan HTTPS ?
    Perbedaan antara HTTP dan HTTPS adalah dari segi keamanan, HTTPS menyediakan lapisan enkripsi tambahan untuk melindungi data yang ditransfer.
  10. Jelaskan Apa itu API? sebutkan contohnya?
    API (Application Programming Interface) adalah kumpulan aturan dan protokol yang digunakan oleh program komputer untuk berkomunikasi satu sama lain. Contohnya: RESTful API: Merupakan pendekatan desain untuk membuat API yang menggunakan metode HTTP (GET, POST, PUT, DELETE) untuk mengelola sumber daya melalui URL.
